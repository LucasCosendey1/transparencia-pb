📋 Visão Geral do Projeto

Transparencia-PB é uma plataforma de transparência governamental construída com Next.js, React e TypeScript. O projeto oferece acesso a informações públicas de forma clara, acessível e interativa, focando em dados orçamentários, contratos, licitações e outros documentos de interesse público do estado da Paraíba.

Objetivos Principais

  • ✅ Disponibilizar dados públicos de forma acessível e transparente
  • ✅ Permitir visualização e filtragem de dados com gráficos interativos
  • ✅ Exportar dados em múltiplos formatos (Excel, PDF, CSV, JSON)
  • ✅ Garantir acessibilidade com suporte a Libras e alto contraste
  • ✅ Proporcionar interface responsiva para desktop e mobile

Editor completo de tabelas para administradores. Permite criar, editar, deletar tabelas e importar dados de Excel/JSON com interface intuitiva de drag-and-drop.

Fluxo de Importação

  1. Selecionar arquivo Excel ou JSON
  2. Indicar se possui cabeçalho (linha de nomes de colunas)
  3. Escolher número da linha com cabeçalho (se aplicável)
  4. Decidir: Substituir todos os dados ou Acrescentar abaixo
  5. Importação confirmada com mensagem de sucesso

🚀 Guia de Início Rápido

Pré-requisitos

  • Node.js 18+ instalado
  • MySQL 8.0+ configurado
  • npm ou yarn como gerenciador de pacotes

Instalação

# 1. Clonar repositório
git clone https://github.com/LucasCosendey1/transparencia-pb.git
cd transparencia-pb

# 2. Instalar dependências
npm install

# 3. Configurar variáveis de ambiente
cp .env.example .env.local
# Editar .env.local com credenciais do MySQL, JWT secret, etc.

# 4. Iniciar servidor de desenvolvimento
npm run dev

# 5. Acessar aplicação
# Abra http://localhost:3000 no navegador
